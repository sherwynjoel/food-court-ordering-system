# Centralized Food Court Ordering System

A multi-tenant system for food courts allowing single-cart ordering from multiple kitchens.

## Structure
-   `backend/`: Node.js (NestJS) API and Real-time Gateway.
-   `admin-web/`: React.js Web Dashboard for Branch and Super Admins.
-   `tablet-app/`: (Planned) Flutter/Android Kiosk App for customers.
